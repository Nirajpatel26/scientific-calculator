# Scientific Calculator

A modern, feature-rich scientific calculator web application with a beautiful gradient UI design.

## Features

### Basic Operations
- Addition, Subtraction, Multiplication, Division
- Percentage calculations
- Negation and absolute value

### Scientific Functions
- **Trigonometric**: sin, cos, tan (in degrees)
- **Logarithmic**: log (base 10), ln (natural log)
- **Exponential**: e^x, x²
- **Mathematical Constants**: π (pi), e (Euler's number)
- **Advanced**: √x (square root), x! (factorial), 1/x (inverse)

### User Experience
- Clean, modern interface with gradient design
- Responsive layout that works on all devices
- Keyboard support for faster calculations
- Real-time display with previous operation history
- Smooth animations and hover effects

## Usage

Simply open `index.html` in your web browser to start using the calculator.

### Keyboard Shortcuts
- `0-9`: Number input
- `+, -, *, /`: Basic operations
- `.`: Decimal point
- `Enter` or `=`: Calculate result
- `Escape`: Clear all
- `Backspace`: Delete last digit

## Technologies Used

- HTML5
- CSS3 (with modern gradients and glassmorphism)
- Vanilla JavaScript (ES6+)

## Installation

1. Clone this repository
2. Open `index.html` in your preferred browser
3. Start calculating!

No build process or dependencies required!
